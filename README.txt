The SchedulingApp is a JavaFX application which provides the ability for a user
to create and manage appointments with customers in multiple timezones. The application
also provides users the ability to create and modify customers and generate reports.

Instructions:
1. Ensure JDK 11 or higher is installed
2. Ensure JavaFX is installed and configured.
3. Import project to IntelliJ IDEA.
4. Run project from IntelliJ - Run -> Run 'Main'.
5. Log into application - Username:test Password:test

Note:
- Language translation is provided in English and French for the Login screen only.
- The fifth-teen minute appointment reminder feature is specific to the user logged in.

- Lambda example #2 can be found at src.Controller.MonthTypeReportController.generateReport().
- The Javadoc index.html file can be found at src.JavaDoc.
